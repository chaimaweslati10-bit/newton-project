## 1. Titre & Description
Ce projet est une application Python avec interface graphique qui permet de résoudre des équations non linéaires en utilisant la méthode de Newton-Raphson.

Il permet de :
- entrer une fonction mathématique
- choisir une valeur initiale
- afficher les étapes de calcul
- visualiser la convergence sur un graphique

---

## 2. Prérequis
Avant de lancer le projet, il faut installer :

- Python 3.x
- pip
- bibliothèques Python :
  - numpy
  - matplotlib
  - tkinter (déjà inclus avec Python)

---

## 3. Installation

Ouvrir le terminal et taper :

pip install numpy matplotlib

---

## 4. Utilisation

Pour lancer le projet, exécuter :

python newton_gui.py

Étapes d’utilisation :
1. Entrer la fonction f(x) (exemple : x**2 - 2)
2. Entrer une valeur initiale x₀
3. Cliquer sur le bouton "Calculer"
4. Observer :
   - les itérations
   - la racine approchée
   - le graphique de la fonction
